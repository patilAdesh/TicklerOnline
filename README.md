## .Features:

- [x] Show tasks list
- [x] Add new task
- [x] Complete task
- [x] Delete task
- [x] Clear complete tasks
- [x] Save on local storage
- [x] Share the task
- [x] Set priority for each task
- [x] Remind about the task
- [x] Add date of each task
- [x] Notifies about the delayed task
- [x] Custom Scrollbar
- [x] User can mark if task is complete
- [x] Responsive
